Eventos criados neste dispositivo

    Como funciona

    1. Crie um evento com o nome da competição.
    2. Compartilhe o link com todo mundo na mesa.
    3. Cada participante cadastra o próprio nome e vai tocando no “+” toda vez que comer um pedaço.
    4. O ranking atualiza automaticamente para todos no evento.

    Perguntas frequentes

    Preciso de login?

    Não. Basta ter o link do evento para visualizar e participar.

    Quantas pessoas podem participar?

    O sistema foi pensado para rodízios pequenos e médios (até algumas dezenas de participantes).

    Posso resetar minha contagem?

    Você pode remover o participante e criar novamente com o mesmo nome para começar do zero.
